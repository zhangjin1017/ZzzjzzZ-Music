## 介绍

一款在线音乐应用，提供歌曲搜索、免费歌曲在线试听、MV播放等功能。


## api服务

[网易云音乐 Node.js API](https://github.com/Binaryify/NeteaseCloudMusicApi)

## 功能

- 播放歌曲
- 搜索歌曲
- 登陆
- 专辑详情、歌单详情
- 歌词
- 主题设置
- 歌手详情
- MV详情
- 用户收藏歌单
